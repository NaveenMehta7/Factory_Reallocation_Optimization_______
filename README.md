# 🏭 Factory Reallocation & Shipping Optimization Recommendation System

## 📌 Project Overview

This project is developed for the Unified Mentor Internship Program.

The main objective is to optimize factory allocation and improve shipping efficiency for Nassau Candy Distributor using Machine Learning. The system analyzes historical sales data, compares different machine learning models, and recommends the most suitable factory for product allocation.

---

## 🎯 Objectives

- Analyze sales and shipping data
- Predict factory performance using Machine Learning
- Compare multiple regression models
- Recommend the best factory for product allocation
- Visualize business insights through an interactive Streamlit dashboard

## 🚀 How to Run the Project

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run Streamlit App

```bash
streamlit run app.py
```
